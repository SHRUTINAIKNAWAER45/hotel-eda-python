# hotel-eda-python
Exploratory Data Analysis of hotel booking data using Python to analyze occupancy, revenue trends, and customer behavior.

Hotel Revenue & Occupancy Analysis (Python EDA)

--Project Overview

This project focuses on analyzing hotel booking data to understand revenue, occupancy trends, and customer behavior across different cities, room categories, and time periods. The goal is to generate actionable insights that can help hotels improve pricing strategy, demand forecasting, and overall performance.
